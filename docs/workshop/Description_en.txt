Is it true that [i] Non Sufficit Orbis [/i] — the world is not enough?

[b] Then give this mod a try! [/b]

[h1] Main Features [/h1]
This mod supports both English and Chinese.

It’s a lightweight UI mod that displays your current world exploration progress above the yield panel, including total plots, explored plots, and the percentage explored.

For changelogs and more, visit GitHub:
[b][url=github.com/jo-xin/PQ2_ExplorationProgress_Civ7Mod]PQ2_ExplorationProgress_Civ7Mod[/url][/b]

[h1] Compatibility [/h1]
Compatible with [b][url=steamcommunity.com/sharedfiles/filedetails/?id=3507072814]Map Trix[/url][/b], which enhances the yield UI.
